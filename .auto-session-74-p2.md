# Task: Stat-var coverage expansion — test multiple stat tokens with Sto/Rcl round-trip

## Context

We are reverse-engineering the TI-84 Plus CE OS ROM. Session 73 proved that Sto_StatVar + Rcl_StatVar works for tMean (token 0x21). The implementation is at:
- Sto_StatVar: `0x09A3BD` — stores OP1 to the stat-var slot derived from A
- Rcl_StatVar: `0x08019F` — reads stat-var slot derived from A into OP1
- CmpStatPtr helper: `0x09A3A5` — converts A (token) → index → memory offset
- Stat-var table base: `statVarsOffset = 0xD0117F`, each slot is 9 bytes (TI FP real format)
- Slot address formula: `0xD0117F + (A * 9)` where A is the stat-var token value
- statsValid: bit 6 of `(IY + 0x09)` = `0xD00089`

The stat-var token values from `ti84pceg.inc` (stat output variable section):
- tStatN = 0x02 (statistics N)
- tXMean = 0x03 (X mean)
- tSumX = 0x04 (sum of X)
- tSumXSqr = 0x05 (sum of X^2)
- tMinX = 0x08 (min X)
- tMaxX = 0x09 (max X)
- tMean = 0x21 (already proven in session 73)

## Your Task

Create `TI-84_Plus_CE/probe-phase25o-statvar-expansion.mjs` that tests the Sto/Rcl round-trip for 4 tokens beyond tMean: **tStatN (0x02), tXMean (0x03), tSumX (0x04), tMinX (0x08)**.

For each token:
1. Cold boot + postInitState (same as 25M probe)
2. Clear the target slot to 0xFF
3. Seed OP1 with a distinct test value (use different values: 10.0, 25.5, 100.0, -3.0)
4. Set statsValid bit 6 at 0xD00089
5. Set `cpu.a = token`, push FAKE_RET, call Sto_StatVar at 0x09A3BD
6. Verify slot bytes match expected encoding
7. Clear OP1 to 0xFF
8. Set `cpu.a = token`, push FAKE_RET, call Rcl_StatVar at 0x08019F
9. Verify round-trip value matches within 1e-9
