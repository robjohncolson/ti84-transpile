## How to disassemble ROM bytes

Read raw bytes from `ROM.rom` and decode manually. The eZ80 instruction set is similar to Z80 but with 3-byte addresses. Key opcodes:
- 0xCD = CALL nn (3-byte address, little-endian)
- 0xC3 = JP nn
- 0xC9 = RET
- 0x18 = JR d (relative jump)
- 0x20 = JR NZ, d
- 0x28 = JR Z, d
- 0xFD prefix = IY operations
- 0xDD prefix = IX operations
- 0xED prefix = extended ops (LDIR=0xB0, LDDR=0xB8, etc.)
- 0xCB prefix = bit operations

For a complete decoder, you can import from `./ez80-decoder.js`:
```js

// decodeInstruction(memoryReadFn, pc) returns { mnemonic, operands, size, ... }
```
