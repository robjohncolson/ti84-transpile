# Task: Add MEM_INIT seeds + build probe calling 0x09DEE0 before ParseInp

## Context
We are reverse-engineering the TI-84 Plus CE OS ROM. ParseInp (0x099914) fails with errNo=0x8E (ErrMemory) even when all heap pointers are manually seeded to correct values. The MEM_INIT routine at 0x09DEE0 does MORE than just seed pointers — it also:
1. Calls 0x08A98F (unknown helper)  
2. Sets bit 7 of (IY+71)
3. Computes flash size via CALL 0x000600 + arithmetic → stores result at 0xD025C5

We believe calling MEM_INIT as a whole (instead of manually seeding individual pointers) may fix ParseInp.

## What to do

### Step 1: Add seeds to transpiler
Edit `scripts/transpile-ti84-rom.mjs` to add these addresses to the `additionalSeeds` array:
- `0x09DEE0` (MEM_INIT entry point)
- `0x08A98F` (helper called by MEM_INIT)

Search for the `additionalSeeds` array in the file — it's a large array of hex addresses. Add these two at the end. They may already be reachable via CFG discovery, but explicit seeds guarantee coverage.

### Step 2: Retranspile
Run: `node scripts/transpile-ti84-rom.mjs`
Verify it completes without error. Note the new block/seed counts from the output.

### Step 3: Build the probe
Create `TI-84_Plus_CE/probe-phase25x-meminit-then-parseinp.mjs` that:

1. Loads the runtime:
```javascript
import { createCPU } from './cpu-runtime.js';
import { createPeripheralBus } from './peripherals.js';
import fs from 'fs';

const ROM_PATH = new URL('./ROM.rom', import.meta.url).pathname.replace(/^\/([A-Z]:)/, '$1');
const TRANSPILED_PATH = new URL('./ROM.transpiled.js', import.meta.url).pathname.replace(/^\/([A-Z]:)/, '$1');

const rom = fs.readFileSync(ROM_PATH);
const peripherals = createPeripheralBus({ timerInterrupt: false });
const cpu = createCPU(peripherals);
cpu.loadROM(rom);

// Load transpiled blocks
const transpiledCode = fs.readFileSync(TRANSPILED_PATH, 'utf8');
const blockFunc = new Function('cpu', 'mem', 'flags', transpiledCode);
blockFunc(cpu, cpu.mem, cpu.flags);
```

2. Sets up minimal state:
```javascript
// IY = 0xD00080 (standard OS value)
cpu.registers.IY = 0xD00080;

// Set up a return sentinel
const FAKE_RET = 0x7FFFFA;
const STACK_TOP = 0xD1A86F;
cpu.registers.SP = STACK_TOP;
cpu.mem[STACK_TOP] = FAKE_RET & 0xFF;
cpu.mem[STACK_TOP + 1] = (FAKE_RET >> 8) & 0xFF;
cpu.mem[STACK_TOP + 2] = (FAKE_RET >> 16) & 0xFF;
```

3. **Phase A**: Calls MEM_INIT at 0x09DEE0:
```javascript
cpu.registers.PC = 0x09DEE0;
let steps = 0;
const MAX_STEPS = 100000;
while (cpu.registers.PC !== FAKE_RET && steps < MAX_STEPS) {
  cpu.step();
  steps++;
}
console.log(`MEM_INIT: ${steps} steps, final PC=0x${cpu.registers.PC.toString(16)}`);
```

4. Dumps pointer state after MEM_INIT:
```javascript
function read24(addr) {
  return cpu.mem[addr] | (cpu.mem[addr+1] << 8) | (cpu.mem[addr+2] << 16);
}
const slots = {
  tempMem: 0xD02587, FPSbase: 0xD0258A, FPS: 0xD0258D,
  OPBase: 0xD02590, OPS: 0xD02593, pTempCnt: 0xD02596,
  pTemp: 0xD0259A, progPtr: 0xD0259D, newDataPtr: 0xD025A0,
  flashSize: 0xD025C5
};
for (const [name, addr] of Object.entries(slots)) {
  console.log(`  ${name} (0x${addr.toString(16)}): 0x${read24(addr).toString(16).padStart(6,'0')}`);
}
```

5. **Phase B**: Now sets up for ParseInp. Places tokenized "2+3\0" at a SEPARATE address from the heap:
```javascript
// Token buffer at 0xD00800 (system scratch, separate from heap)
const TOKEN_BUF = 0xD00800;
const tokens = [0x32, 0x70, 0x33, 0x00]; // "2", "+", "3", null
for (let i = 0; i < tokens.length; i++) cpu.mem[TOKEN_BUF + i] = tokens[i];

// Point OPS to token buffer (parser reads from OPS)
cpu.mem[0xD02593] = TOKEN_BUF & 0xFF;
cpu.mem[0xD02594] = (TOKEN_BUF >> 8) & 0xFF;
cpu.mem[0xD02595] = (TOKEN_BUF >> 16) & 0xFF;
```

6. Sets up errSP for error handler (critical for ParseInp):
```javascript
// PushErrorHandler frame at stack area
const ERR_FRAME_ADDR = STACK_TOP - 20;
cpu.registers.SP = ERR_FRAME_ADDR;

// Push new return sentinel for ParseInp
const PARSE_RET = 0x7FFFF0;
cpu.mem[ERR_FRAME_ADDR] = PARSE_RET & 0xFF;
cpu.mem[ERR_FRAME_ADDR + 1] = (PARSE_RET >> 8) & 0xFF;
cpu.mem[ERR_FRAME_ADDR + 2] = (PARSE_RET >> 16) & 0xFF;

// Seed errSP to our frame
cpu.mem[0xD025A6] = ERR_FRAME_ADDR & 0xFF;
cpu.mem[0xD025A7] = (ERR_FRAME_ADDR >> 8) & 0xFF;
cpu.mem[0xD025A8] = (ERR_FRAME_ADDR >> 16) & 0xFF;

// Seed FPSbase for error recovery
// (use same value as FPS from MEM_INIT — read it back)
const fpsVal = read24(0xD0258D);
cpu.mem[0xD0258A] = fpsVal & 0xFF;
cpu.mem[0xD0258B] = (fpsVal >> 8) & 0xFF;
cpu.mem[0xD0258C] = (fpsVal >> 16) & 0xFF;
```

7. **Phase B execution**: Calls ParseInp:
```javascript
cpu.registers.PC = 0x099914;
const errNoAddr = 0xD025A9; // errNo location
cpu.mem[errNoAddr] = 0x00; // clear errNo

let parseSteps = 0;
const PARSE_MAX = 500000;
const pcHistory = [];
while (cpu.registers.PC !== PARSE_RET && cpu.registers.PC !== FAKE_RET && parseSteps < PARSE_MAX) {
  if (parseSteps < 200 || parseSteps % 10000 === 0) {
    pcHistory.push({ step: parseSteps, pc: cpu.registers.PC });
  }
  cpu.step();
  parseSteps++;
}
console.log(`\nParseInp: ${parseSteps} steps, final PC=0x${cpu.registers.PC.toString(16)}`);
console.log(`errNo: 0x${cpu.mem[errNoAddr].toString(16).padStart(2,'0')}`);

// Dump OP1
const op1 = [];
for (let i = 0; i < 9; i++) op1.push(cpu.mem[0xD005F8 + i].toString(16).padStart(2,'0'));
console.log(`OP1: [${op1.join(' ')}]`);

// Dump pointer state post-ParseInp
console.log('\nPost-ParseInp pointer state:');
for (const [name, addr] of Object.entries(slots)) {
  console.log(`  ${name}: 0x${read24(addr).toString(16).padStart(6,'0')}`);
}

// First 20 PC entries
console.log('\nFirst 20 PC trace entries:');
pcHistory.slice(0, 20).forEach(e => console.log(`  step ${e.step}: 0x${e.pc.toString(16).padStart(6,'0')}`));
```

8. Print verdict:
```javascript
const errNo = cpu.mem[errNoAddr];
if (cpu.registers.PC === PARSE_RET && errNo === 0x00) {
  console.log('\n✅ PASS: ParseInp returned successfully with no error');
} else if (cpu.registers.PC === PARSE_RET || cpu.registers.PC === FAKE_RET) {
  console.log(`\n❌ PARTIAL: ParseInp returned but errNo=0x${errNo.toString(16).padStart(2,'0')}`);
} else {
  console.log(`\n❌ FAIL: ParseInp did not return (stuck at 0x${cpu.registers.PC.toString(16)}, ${parseSteps} steps)`);
}
```
