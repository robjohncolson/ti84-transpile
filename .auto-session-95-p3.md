## eZ80 decode approach
Import the decoder:
```js
import { decodeInstruction } from './ez80-decoder.js';
```
If that doesn't work, fall back to manual byte decode. Key patterns:
- `CD xx xx xx` = CALL (3 bytes addr, little-endian, ADL mode)
- `C3 xx xx xx` = JP
- `C9` = RET
- `FD CB nn xx` = IY+offset bit ops
- `32 xx xx xx` = LD (addr),A
- `3A xx xx xx` = LD A,(addr)
- `ED B0` = LDIR, `ED B8` = LDDR
