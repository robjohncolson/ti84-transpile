# Task: Phase 25P — Stat-var expansion (more tokens + edge cases)

## Context
You are working on a TI-84 Plus CE ROM transpilation project. We've validated Sto/Rcl round-trip for 5 stat-var tokens so far: tMean(0x21), tStatN(0x02), tXMean(0x03), tSumX(0x04), tMinX(0x08). The slot formula is `0xD0117F + (A × 9)` where A is the stat-var token. Store impl = 0x09A3BD, Recall impl = 0x08019F.

## Goal
Write a probe `TI-84_Plus_CE/probe-phase25p-statvar-extended.mjs` that tests 4 MORE stat-var tokens plus edge cases:
1. tMaxX(0x09) = 999.0 (large value)
2. tCorr(0x12) = 0.95 (fractional)
3. tQ1(0x14) = 0.0 (zero value — edge case)
4. tQ3(0x15) = -100.5 (negative fractional)

## Template
Use `TI-84_Plus_CE/probe-phase25o-statvar-expansion.mjs` as template. The pattern is:
1. Cold boot + postInitState
2. For each test case:
   a. Set statsValid bit 6 at 0xD00089: `mem[0xD00089] |= 0x40`
   b. Write the value to OP1 via `writeReal(memWrap, OP1_ADDR, value)`
   c. Set `cpu._a = token` (the stat-var token byte)
   d. Push FAKE_RET, call Sto_StatVar impl at 0x09A3BD
   e. Verify the slot at `0xD0117F + (token * 9)` contains the stored bytes
   f. Clear OP1 to 0xFF
   g. Set `cpu._a = token` again
   h. Push FAKE_RET, call Rcl_StatVar impl at 0x08019F
   i. Read OP1 via `readReal` and compare to expected value

Key details:
- OP1 = 0xD005F8
- statsValid flag: bit 6 of byte at 0xD00089
- IY = 0xD00080 (must be set before each call — postInitState sets it)
- Between calls, reset: `cpu.halted = false; cpu.iff1 = 0; cpu.iff2 = 0; cpu.madl = 1; cpu._iy = 0xD00080; cpu.f = 0x40; cpu._ix = 0xD1A860; cpu.sp = STACK_RESET_TOP - 12; mem.fill(0xFF, cpu.sp, cpu.sp + 12);`
- `mem.fill(val, start, END)` — third arg is END not length!
- Timer IRQ disabled: `createPeripheralBus({ timerInterrupt: false })`
- FAKE_RET = 0x7FFFFE
- INSN_BUDGET = 200000 per call
- TOLERANCE = 1e-6
