## Exact Procedure

### Boot Sequence (copy from browser-shell pattern)
1. Load ROM: `fs.readFileSync('ROM.rom')` into `new Uint8Array(0x1000000)`
2. Load transpiled blocks: `await import('ROM.transpiled.js')` → `PRELIFTED_BLOCKS`
3. Create peripherals: `createPeripheralBus({ pllDelay: 2, timerInterrupt: false })`
4. Create executor: `createExecutor(BLOCKS, mem, { peripherals: perph })`
5. Cold boot: `ex.runFrom(0x000000, 'z80', { maxSteps: 20000, maxLoopIterations: 32 })`
6. Kernel init: reset CPU, `c.sp = 0xD1A87E - 3`, run from `0x08C331` in `adl`, 100000 steps
7. Post-init: `c.mbase = 0xD0; c._iy = 0xD00080; c._hl = 0`, run from `0x0802B2` in `adl`, 100 steps
8. MEM_INIT: set `c.madl=1, c.mbase=0xD0, c._iy=0xD00080, c.f=0x40, c._ix=0xD1A860`, sp=0xD1A87E-12, push FAKE_RET=0x7FFFFE, run from 0x09DEE0, catch FAKE_RET to confirm completion
9. CreateReal(Ans): set OP1 = `[0x00, 0x72, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]` at 0xD005F8, push FAKE_RET, run from 0x08097C in adl, catch FAKE_RET. Save post-CreateReal OPS/FPS/FPSBASE values.

### BufInsert Phase
Constants:
```
EDIT_TOP    = 0xD02437  (24-bit pointer)
EDIT_CURSOR = 0xD0243A
EDIT_TAIL   = 0xD0243D
EDIT_BTM    = 0xD02440
BUFINSERT   = 0x05E2A0
BUF_START   = 0xD00A00
BUF_END     = 0xD00B00
```

1. Initialize gap-buffer pointers:
   - write24(EDIT_TOP, BUF_START)
   - write24(EDIT_CURSOR, BUF_START)
   - write24(EDIT_TAIL, BUF_END)
   - write24(EDIT_BTM, BUF_END)
2. Clear buffer: `mem.fill(0x00, BUF_START, BUF_END)`
3. For each token in [0x32, 0x70, 0x33] ("2", "+", "3"):
   - Reset CPU: `c.halted=false, c.iff1=0, c.iff2=0, c.madl=1, c.mbase=0xD0, c._iy=0xD00080, c.f=0x40, c._ix=0xD1A860`
   - Set sp = 0xD1A87E - 12, fill 12 bytes of 0xFF, push FAKE_RET
   - Set `c._de = token` (token in E register, D=0 for single-byte)
   - Run from 0x05E2A0 in adl, maxSteps 10000, catch FAKE_RET
   - Verify it returned

### Parse Phase
After BufInsert completes for all 3 tokens:
1. Read cursor position: `read24(EDIT_CURSOR)`
2. Compute preGapLen = cursor - BUF_START
3. Set parse pointers:
   - write24(0xD02317, BUF_START)  // begPC
   - write24(0xD0231A, BUF_START)  // curPC
   - write24(0xD0231D, BUF_START + preGapLen - 1)  // endPC
4. Restore OPS/FPS/FPSBASE to post-CreateReal values
5. Clear OP1: `mem.fill(0x00, 0xD005F8, 0xD005F8 + 9)`
6. Set up error frame:
   - sp = 0xD1A87E - 12, fill 0xFF
   - push FAKE_RET (0x7FFFFE)
   - errBase = sp - 6
   - write24(errBase, 0x7FFFFA)  // ERR_CATCH
   - write24(errBase + 3, 0)
   - write24(0xD008E0, errBase)  // errSP
   - mem[0xD008DF] = 0x00  // errno
7. Run ParseInp from 0x099914 in adl, maxSteps 1500000, catch FAKE_RET or ERR_CATCH

### Verification
Read OP1 (9 bytes at 0xD005F8). Expected for 5.0: `[0x00, 0x80, 0x50, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]`

Print PASS/FAIL based on OP1 match. Also print:
- Buffer contents after BufInsert (hex dump of BUF_START to BUF_START+16)
- Cursor position after BufInsert
- Parse pointer values (begPC, curPC, endPC)
- OP1 hex dump
- Parsed value as float (decode BCD: sign from byte 0 bit 7, exponent = byte1 - 0x80, mantissa = BCD digits from bytes 2-8)

## Helper: write24 / read24
```javascript
function write24(addr, val) {
  mem[addr] = val & 0xFF;
  mem[addr+1] = (val >> 8) & 0xFF;
  mem[addr+2] = (val >> 16) & 0xFF;
}
function read24(addr) {
  return mem[addr] | (mem[addr+1] << 8) | (mem[addr+2] << 16);
}
```
