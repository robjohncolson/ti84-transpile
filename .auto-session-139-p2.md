- E_Domain (0x84) at gcd handler caused by: FPS=EMPTY (0 bytes), OP1=garbage, OP2=malformed

## BCD format reference
- 9 bytes: [type, exponent, d1d2, d3d4, d5d6, d7d8, d9d10, d11d12, d13d14]
- type 0x00 = positive real
- exponent = 0x80 + power_of_10 (0x81 = 10^1, 0x80 = 10^0)
- 12.0 = [0x00, 0x81, 0x12, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]
- 8.0 = [0x00, 0x80, 0x80, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]
- 4.0 = [0x00, 0x80, 0x40, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]
