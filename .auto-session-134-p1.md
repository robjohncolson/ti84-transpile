## Context
This is a TI-84 Plus CE ROM transpilation project. The transpiler at `scripts/transpile-ti84-rom.mjs` lifts eZ80 ROM bytes into JavaScript. Seeds are addresses that tell the transpiler "start decoding here." Missing seeds = missing blocks = unreachable code at runtime.

Session 133 traced gcd(12,8) through ParseInp and found the FP dispatch path has 7 missing blocks that prevent the floating-point handler table from working. The category byte at 0xD0060E is written but then cleared before dispatch because the dispatch table entries themselves have no blocks.
