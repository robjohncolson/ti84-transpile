## Key addresses

- 0xD005F8 = OP1, 0xD00603 = OP2, 0xD0060E = OP3, 0xD00624 = OP5
- 0x07F8A2 = OP1→OP4, 0x07F8B6 = OP4→OP2

## TI floating point format
- Byte 0: sign/type (0x00=positive real, 0x80=negative real)
- Byte 1: exponent (biased by 0x80, so 0x80=10^0, 0x81=10^1, 0x83=10^3)
- Bytes 2-8: BCD mantissa (e.g., 12 00 00 00 00 00 00 = 0.12 × 10^exp)
- So [00, 81, 12, 00...] = +0.12 × 10^1 = 1.2... wait, [00, 81, 12, 00...] = 12.0
- And [00, 83, 12, 00...] = 1200.0
