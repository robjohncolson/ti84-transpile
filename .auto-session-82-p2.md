## eZ80 ADL encoding reference
Key opcodes (all in ADL mode, 24-bit addresses):
- `21 xx yy zz` = LD HL, imm24
- `22 xx yy zz` = LD (imm24), HL
- `2A xx yy zz` = LD HL, (imm24)
- `CD xx yy zz` = CALL imm24
- `C3 xx yy zz` = JP imm24
- `C9` = RET
- `ED 43 xx yy zz` = LD (imm24), BC
- `ED 53 xx yy zz` = LD (imm24), DE
- `AF` = XOR A
- `3E xx` = LD A, imm8
- `32 xx yy zz` = LD (imm24), A
- `CB xx` = bit/rotate ops
- `FE xx` = CP imm8
- `18 xx` = JR offset
- `20 xx` = JR NZ, offset
- `28 xx` = JR Z, offset

Some 2-byte prefix ops:
- `DD` prefix = IX variants
- `FD` prefix = IY variants  
- `ED` prefix = extended ops
