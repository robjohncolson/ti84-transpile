Phase 50.2 — Probe 0x081670 (STAT/MATRIX editor renderer) with various menu_mode values to find a home-screen variant.

CONTEXT:
- TI-84 ROM transpilation project, dir already set.
- 0x081670 was identified in Phase 31 as the STAT/MATRIX editor render function. It draws 1534 pixels in a 155x308 grid layout. Phase 49 re-verified it draws 49908 cells with corrected (boot-only) template. Bbox r55-210 c0-319.
- The (0xd007e0) byte is a "menu mode" state that 219 ROM sites read. Different values produce different screens. Phase 48 enumerated 12 dispatchers that branch on (0xd007e0)==0x40, but none of them rendered the home screen.
- Phase 25's home screen hunt failed because the post-boot callback at 0x015AD9 turned out to be a busy-wait delay loop (NOT a screen renderer).
