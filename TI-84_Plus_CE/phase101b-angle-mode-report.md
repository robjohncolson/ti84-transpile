## Post-Probe Deep Investigation

After the automated scan, manual tracing and dynamic probing revealed:

### Key Finding: RAD/DEG text is NOT in r0-16

The TI-84 CE home screen status bar (r0-16) does NOT contain angle mode text ('RAD'/'DEG').
The confirmed contents of r0-16 are:
- **r0-2**: all white (blank)
- **r3-6**: status ICON pixels at cols ~146-150 and ~306-310 (drawn by `0x0a3301`)
- **r7-16**: all white (blank)

The 'Radian'/'Degree'/'Normal'/'Float' text lives exclusively in the HOME ROW at **r17-34**,
rendered by the stage-3 composite (`0x0a29ec (= RStrCurRow)` with mode buffer seed at `0xD020A6`).

### False Positive Explanation

`0x0a2edc` registered 764 fg pixels in the initial scan because:
- The VRAM baseline already has 5440 drawn pixels (all white) from stage 1
- The scan compared fg counts against the baseline snapshot, not against a sentinel baseline
- The bytes at `0x0a2edc` are ROM string table data ('RAM', 'ROM', 'ID', 'SELECT', 'TRANSMIT'...)
  being mis-decoded as Z80 instructions. They don't actually write to VRAM.

### Mode String Architecture

The mode string table is at ROM `0x0a044f`:
- `Radian` at `0x0a044f`, `Degree` at `0x0a0457`, `Normal` at `0x0a045f`, `Float` at `0x0a0471`
- These are copied to RAM buffer `0xD006C0` (26 bytes per mode, 10+ rows)
- The home row renderer (`0x0a29ec (= RStrCurRow)`) reads from `0xD020A6` (the current composite mode string)
- The status bar renderer (`0x05e71e`) exists but only updates RAM mode buffers; it does NOT
  write pixels to VRAM r0-16

### Conclusion

**There is no angle mode text renderer writing to r0-16.** The Phase 99 golden regression
(r17-34 with 'Normal Float Radian') is the correct and complete mode display. The status bar
(r0-16) contains only: white background + 2 colored status dot icons at fixed pixel positions.

**Next step**: The composite is complete as-is. Phase 102 should verify the golden regression
still passes after Phase 101A's `0x0a3301` icon pass, or investigate what the `D020A6` angle
mode flag controls (Radian vs Degree) to confirm the mode string seed is appropriate.
