## Context
You are working on a TI-84 Plus CE ROM transpilation project at `C:/Users/rober/Downloads/Projects/school/ti84-transpile`. The project lifts eZ80 machine code into JavaScript.

Session 102 discovered that the VAT walker loop at 0x082745 is infinite because OPBase at 0xD02590 is zeroed after MEM_INIT runs. The loop does `LD BC,(0xD02590); SBC HL,BC; RET C` — exits when HL < OPBase. With OPBase=0, the loop never exits.

MEM_INIT (entry 0x09DEE0) originally sets OPBase=0xD3FFFF (per session 82 disassembly), but subsequent operations zero it. The fix: re-seed OPBase AFTER MEM_INIT before running the ENTER handler.
