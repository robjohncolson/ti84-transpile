# Session 159 P1 — Disassemble 0x07FD4A (validity check routine)

## Context

You are working on a TI-84 Plus CE ROM transpilation project at `C:/Users/rober/Downloads/Projects/school/ti84-transpile`. The ROM is lifted from eZ80 machine code to JavaScript. The current investigation focuses on the FP gcd bug where `gcd(12,8)` returns E_Domain instead of 4.

The normalization function at 0x07CA48 is called before FPDiv. At 0x07CA48, the FIRST instruction is `CALL 0x07FD4A` (a "validity check"), followed by `RET Z` at 0x07CA4C. If the validity check sets Z flag, normalization is SKIPPED entirely via RET Z.

Session 158 P3 tested this: the validity check does NOT set Z for inputs with exponent >= 0x80. But we need to understand WHAT it actually checks and whether it could set Z for the specific gcd inputs (exp 0x83, mantissa 1200).

## Task

Build a probe file `TI-84_Plus_CE/probe-phase159-07fd4a-disasm.mjs` that:

### Part A: Static ROM disassembly of 0x07FD4A
- Read ROM bytes starting at 0x07FD4A and disassemble using `decodeInstruction` from `ez80-decoder.js`
- Disassemble at least 30 instructions or until you hit a RET/JP that exits the function
- Print each instruction with address, hex bytes, and mnemonic
- Note any conditional returns, flag-setting instructions, or memory accesses

### Part B: Dynamic trace of 0x07FD4A with gcd-relevant inputs
- Set up a full execution environment (see boilerplate below)
- Seed OP1 with BCD 1200 (type=0x00, exp=0x83, mantissa=[12 00 00 00 00 00 00])
- Call 0x07FD4A directly
- Log ALL register state (A, F, BC, DE, HL) at entry and exit
- Specifically log: Z flag at exit, A register at exit, what memory addresses were read/written
- Use onBlock to trace every block visited

### Part C: Compare with different inputs
- Test 0x07FD4A with:
  - OP1 = 0.5 (exp 0x7F, mantissa [50 00 00 00 00 00 00])  
  - OP1 = 0.0 (exp 0x80, mantissa all zeros)
  - OP1 = 1.0 (exp 0x80, mantissa [10 00 00 00 00 00 00])
- For each: log Z flag at exit, A at exit
- We want to find: does 0x07FD4A EVER set Z? Under what conditions?
