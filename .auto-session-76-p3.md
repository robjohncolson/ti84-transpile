# Task: InvOP1S FP primitive probe

## Context
Session 75 validated InvOP1Sc at 0x07CA02 (negates OP1 by flipping sign byte). The related entry InvOP1S at 0x07CA06 is 4 bytes later and may do the same thing or take the absolute value. We want to test it.

From the jump table (TI-84_Plus_CE/phase25h-a-jump-table.json):
- InvOP1Sc: JT slot impl = 0x07CA02
- InvOP1S: JT slot impl = 0x07CA06

Session 75 proved InvOP1Sc(7.0) → -7.0 (sign byte 0x00→0x80). InvOP1S is 4 bytes later, suggesting it skips some prefix instruction.

Create TWO files:
1. `TI-84_Plus_CE/probe-phase25q-invop1s.mjs` — dynamic execution probe
2. `TI-84_Plus_CE/phase25q-invop1s-report.md` — generated report
