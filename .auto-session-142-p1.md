# Auto-Session 142 P1 — OneVar 8/8 with correct entry point 0x0A9319

## Context
The TI-84 ROM transpilation project lifts eZ80 machine code into JavaScript. OneVar computes 1-variable statistics on a list. Session 140 got 6/8 stat results correct (n, xMean, sumX, sumX2, Sx, sigmaX) but minX and maxX remained zero. Session 141 found the root cause: we were using entry point 0x0A9325 which does `SET 5,(IY+9)` — this makes the min/max routine at 0x0AA6FA return immediately (`BIT 5,(IY+9); RET NZ`). The FULL entry at 0x0A9319 does `RES 5,(IY+9)` first (clears the skip flag), enabling min/max computation.

3. **Increases the step budget** to 200,000 (the full entry does more work including sort for min/max)
4. **Checks all 8 stat var slots** after completion:
