# Task: Static disassembly of OP1toOP2 compound function 0x07C740-0x07C780

## Context

The TI-84 CE ROM has a compound function region at 0x07C740-0x07C780 with MULTIPLE entry points:
- 0x07C747: copy OP1→OP2 + normalize + InvOP1S (negate) + FPAdd
- 0x07C755: copy OP1→OP2 + FPAdd (NO normalize)
- 0x07C75B: load 1.0 into OP1 + FPAdd (?)
- 0x07C767: + FPSub (?)
- 0x07C76D: 1.0 + FPSub (?)
- 0x07C771: FPSub entry
- 0x07C77F: FPAdd entry
- 0x07C783: FPAdd+4

Session 160 P3 identified these from a static scan. We need a complete byte-by-byte disassembly of the entire 0x07C740-0x07C790 range (80 bytes) to:
1. Map every instruction and its effect
2. Understand which entry points are for add, subtract, multiply pre-processing, etc.
3. Determine the mathematical intent of each entry variant
