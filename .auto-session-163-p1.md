## eZ80 instruction encoding reference (key opcodes)
- CALL nn = CD nn24 (4 bytes in ADL mode)
- JP nn = C3 nn24 (4 bytes)
- JR e = 18 e8 (2 bytes, signed offset from next instruction)
- JR NZ,e = 20 e8
- JR Z,e = 28 e8
- LD A,(nn) = 3A nn24
- CP n = FE n8
- RET = C9
- RET Z = C8
- RET NZ = C0
