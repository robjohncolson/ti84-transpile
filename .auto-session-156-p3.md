# Task: Track OP3 usage after OP1→OP3 copy in gcd algorithm

## Context
This is a TI-84 Plus CE ROM transpilation project. The ROM is lifted to JavaScript and executed in Node.js.

During gcd(12,8), at step 0x068D8D the algorithm calls 0x07F95E which copies OP1→OP3 (backup). OP3 is at address 0xD00624 (9 bytes). We need to know: is OP3 read or modified later in the gcd algorithm? Session 155 asked this but didn't answer it.

The gcd handler runs for 1442 steps total. The OP1→OP3 copy happens early (around step 52-57 per session 155's trace). If OP3 is used later, it could be important for understanding the algorithm's expected behavior.

OP1 = 0xD005F8 (9 bytes), OP2 = 0xD00603 (9 bytes), OP3 = 0xD00624 (9 bytes).

## Deliverables
Build a probe `TI-84_Plus_CE/probe-phase156-op3-track.mjs` that:

1. Loads the transpiled ROM and runtime
2. Sets up gcd(12,8) computation:
   - Cold boot the OS, run MEM_INIT
   - Set up FPS with 12.0 and 8.0 as the two operands
   - Call the gcd handler at 0x068D3D (or via the dispatch chain 0x06859B)
3. Uses a write-watchpoint on OP3 (0xD00624, 9 bytes) and a read-watchpoint to detect any access
4. Logs every step where OP3 bytes change, or where the PC is in a block that references address 0xD00624
5. Also log OP3's final value when the algorithm completes or hits E_Domain

For read detection: since we can't easily intercept reads in the transpiled JS, instead:
- Log OP3 at each block
- Also do a ROM binary scan for all instructions that reference addresses 0xD00624-0xD0062C (OP3 range) — specifically `LD` instructions loading from or storing to those addresses

## ROM binary scan
Also scan the ROM binary for references to OP3:
```javascript
import { readFileSync } from 'fs';
const rom = readFileSync('TI-84_Plus_CE/ROM.rom');

// Search for 3-byte LE address 0xD00624 = bytes [24, 06, D0]
// and 0xD00625, 0xD00626, etc.
for (let i = 0; i < rom.length - 2; i++) {
    if (rom[i] === 0x24 && rom[i+1] === 0x06 && rom[i+2] === 0xD0) {
        console.log(`OP3 ref at ROM offset 0x${i.toString(16)}`);
    }
}
```

Print all findings to stdout.
