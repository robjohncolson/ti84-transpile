# Task: ParseInp deep-dive — disassemble helper subroutines to find global-pointer contract

## Context

We are reverse-engineering the TI-84 Plus CE OS ROM (eZ80, 4MB at `TI-84_Plus_CE/ROM.rom`). The ROM is loaded into a 16MB `Uint8Array` and executed via a transpiled JS runtime. We have a working probe framework.

Session 73 attempted to call `ParseInp` (expression parser) at `0x099914` but FAILED because it's pointer-driven, not register-driven. First bytes at 0x099914: `af 32 be 22 d0 cd 81 9b 09 fd cb 1f 9e cd 81 9b 09 cd 18 9b 09 c1 cd ed be 09 ...` = `xor a; ld (0xD022BE),a; call 0x099B81; bit 3,(iy+0x1F); call 0x099B81; call 0x099B18; pop bc; call 0x09BEED; ...`

The probe seeded 8 global pointer slots (tempMem, FPSbase, FPS, newDataPtr, 0xD02317, 0xD0231A, 0xD0231D, 0xD007FA, 0xD008E0) to userMem=0xD1A881 but none was picked up. Execution looped at 0x082be2/0x084711/0x084716/0x08471b for 200K steps.

1. **Disassembles** the three helper subroutines that ParseInp calls early:
   - `0x099B81` — called twice
   - `0x099B18` — called once
   - `0x09BEED` — called once

   For each, dump 64 bytes of raw ROM starting at the address (from ROM.rom), then use the eZ80 decoder to disassemble them. Import the decoder:

4. **Write a report** to `TI-84_Plus_CE/phase25o-parseinp-helpers-report.md` containing:

## Template

Use this pattern for reading ROM bytes and decoding:
```js
import fs from 'node:fs';
import path from 'node:path';
import { fileURLToPath } from 'node:url';
import { decodeInstruction } from './ez80-decoder.js';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const romBytes = fs.readFileSync(path.join(__dirname, 'ROM.rom'));

function disasmRange(startAddr, maxBytes) {
  const lines = [];
  let pc = startAddr;
  const end = startAddr + maxBytes;
  while (pc < end) {
    const insn = decodeInstruction(romBytes, pc, true);
    if (!insn || insn.length === 0) { lines.push(`${hex(pc)}: ??? (decode failed)`); break; }
    const bytes = [];
    for (let i = 0; i < insn.length; i++) bytes.push(romBytes[pc + i].toString(16).padStart(2, '0'));
    const operandsStr = insn.operands ? insn.operands.join(', ') : '';
    lines.push(`${hex(pc)}: ${bytes.join(' ').padEnd(15)} ${insn.mnemonic} ${operandsStr}`);
    pc += insn.length;
    if (['RET', 'RETI', 'RETN'].includes(insn.mnemonic) && !operandsStr) break;
    if (['JP', 'JR'].includes(insn.mnemonic) && !operandsStr.includes(',')) break; // unconditional jump
  }
  return lines;
}

function hex(v, w = 6) { return '0x' + v.toString(16).padStart(w, '0'); }
```
